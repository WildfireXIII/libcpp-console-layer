clear
void clear()

Clears the console buffer.
getConsoleSize
void getConsoleSize(int* width, int* height)

Sets passed width and height variables to the current buffer dimensions. NOTE: Buffer dimensions, not console window dimensions!

Input Parameters

int* width - POST: Will point to the number of buffer columns.

int* height - POST: Will point to the number of buffer rows.

Output

The passed width and height pointers will point to the dimensions of the console buffer.
getCursorPos
void getCursorPos(int* x, int* y)

Sets passed x and y variables to the current cursor position.

Input Parameters

int* x - POST: Will point to the current cursor column.

int* y - POST: Will point to the current cursor row.

Output

The passed x and y pointers will point to the current coordinates of the cursor.
getNextKeyPress
int getNextKeyPress()

Puts the console in raw input mode until a key is hit, then returns that key code. (NOTE: This function hangs program operation until a key is hit.)

Output

Returns the key code for pressed key. Note that this varies per operating system. For Windows, check MSDN Virtual-Key Codes. Linux is just weird....

void restoreCursorPos()

Moves cursor to previously stored position. (MUST have previously stored position!) NOTE: This is buffer position, so keep in mind that if you've reached the end of the buffer and are printing out more things, the position will not be on the same CHARACTER as before, only the same place in the buffer.)
